Game Controller Guide

  • Left Analogue Stick Up: Throttle
  • Left Analogue Stick Left/RightYaw (horizontal) rotation
  • Right Analogue Stick Up/DownPitch (forward/back) rotation
  • Right Analogue Stick Left/RightRoll (left/right) rotation


Keyboard controller Guide

  • W key:Throttle
  • A&D key:Yaw rotation
  • Up&Down key:Pitch rotation
  • Left&Right key:Roll rotation
